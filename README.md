# Stock-Analysis

Overview of Project


Help Steve analyze a handful of green energy stocks including DAQO stock. 


Purpose and Background 


Provide a tool that allows end users to gain insight on past stock history to determine if it is worthwhile to invest. 


Results 


Run time of the original versus the refactored was exponentially quicker. In addition the program is much easier for end users to navigate and use analytics tool.

Summary: 

In general the advantages and disadvantages of refactoring code and this also pertains to the original and refactored VBA script

  Advantages:

  -	Improves code writing when refactoring code 
  -	Improves the design side of software when refactoring code
  -	Makes the software easier to understand when refactoring code
  -	Refactoring allows for the program to run faster.
  -	In addition, refactoring helps find bugs.

  Disadvantages 

  -	Refactoring could introduce new bugs and errors
  -	Not specific to this project but if refactoring exceeds budget plans
  -	Not specific to this project but if refactoring becomes too time consuming
